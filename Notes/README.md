# Notes

This directory contains class notes and study materials organized by type.

## Structure

### Class Notes/
Contains weekly class notes in Markdown format:
- Week 1 through Week 14 (individual files for each week)
- Notes cover topics discussed in class, including concepts, definitions, and examples

### Study Guide/
Contains study materials for design patterns:
- **Design_Patterns_Explanations.md**: Comprehensive explanations of design patterns
- **Design_Patterns_Questions.md**: Questions and practice materials for design patterns

### temp/
Temporary folder for work-in-progress or draft notes.
