# Assignments

This directory contains all course assignments, organized into two subdirectories: Individual and Group assignments.

## Structure

### Individual/
Contains individual assignments focused on specific design patterns:
- **Assignment 1 - state**: State pattern implementation with diagrams and LaTeX documentation
- **Assignment 2 - Composite**: Composite pattern with documentation and LaTeX source files
- **Assignment 3 - Command**: Command pattern implementation with UML diagrams (PlantUML) and LaTeX documentation
- **Assignment 4 - Discuss Pattern**: Discussion assignment covering multiple patterns (Facade, Mediator, Strategy)

Each assignment typically includes:
- Documentation and instructions
- Source code (where applicable)
- LaTeX source files and compiled PDFs
- Diagrams (UML, activity diagrams, etc.)

### Group/
Contains group participation assignments and projects:
- **Participation 1**: Singleton Pattern in Real Systems
- **Participation 2**: Design Pattern Specifications & Comparisons
- **Participation 3**: State Pattern implementation in C# (.NET) - Daily Operations Dashboard
- **Participation 4**: Interpreter Pattern implementation in Java with decision tables
- **Prototype**: Prototype pattern implementation in JavaScript with model strategies
