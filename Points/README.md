# Points

This directory contains tracking of participation points earned throughout the course.

## Structure

### points.md
Markdown file tracking all participation points with:
- Date of earning points
- Question or activity that earned points
- Answer or contribution made
- Point value awarded
- Indication of individual vs. group points

Total points tracked: 11 points

### Latex/
Contains LaTeX source files for generating a formatted PDF version of the points document:
- **points.tex**: Main LaTeX source file
- **points.pdf**: Compiled PDF output
- Build artifacts (.aux, .log, .synctex.gz, etc.)
